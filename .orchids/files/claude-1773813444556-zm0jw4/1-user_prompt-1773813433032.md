# Summary

A warm, high-end editorial portfolio design with a luxury boutique feel, utilizing sophisticated serif typography, sepia image treatments, and subtle micro-interactions like multi-layered text echoes and smooth scale transitions.

# Style

The style is 'Editorial Luxury'. It uses a warm-toned neutral palette: cream (#faf8f3), deep espresso (#2a2219), and muted gold (#d4a574). Typography pairings include Boska for headings (high-contrast serif) and Plus Jakarta Sans/Satoshi for body and UI elements (clean sans-serif). Visuals are characterized by high-radius rounded corners (32px-40px), very soft shadows (#2a2219 at 5% opacity), and sepia-filtered images that reveal color on hover.

## Spec

Create a design system with an 'Editorial Luxury' aesthetic. Colors: Background #faf8f3, Text #2a2219, Accents #d4a574, Secondary Text #8c8279. Typography: Use 'Boska' serif for H1-H3 (weight 500-900, italic for emphasis) and 'Plus Jakarta Sans' for body and utility (weight 400-700). UI elements should have 2rem (32px) to 2.5rem (40px) border-radii. Layout borders: 1px solid rgba(42, 34, 25, 0.08). Gradients: Use #fdfcf9 to #f5f1e8 for subtle section shifts. Animation: All image hover effects must use cubic-bezier(0.4, 0, 0.2, 1) over 1s, transitioning filter: sepia(0.3) contrast(0.95) to filter: sepia(0) contrast(1.05) and scale(1.08). Shadows should be extremely subtle (shadow-2xl with 5% opacity of #2a2219).

# Layout & Structure

A vertical-scrolling single-page architecture featuring a fixed blurred navigation, high-impact hero sections with large-scale typography, and a mix of symmetrical and asymmetrical grids for showcasing services and case studies.

## Fixed Navigation

Height 80px, fixed to top. Background #faf8f3/80 with backdrop-blur-xl. Logo on left in Boska italic bold. Centered menu links in all-caps Satoshi, 11px, tracking 0.25em. Right-side 'Inquiry' button: Pill-shaped, background #2a2219, text #faf8f3, all-caps, 10px, 0.2em tracking. Hover state: background #d4a574.

## Hero Section

Full viewport height. Centered vertical stacking. Top label: 'Applied Intelligence' in 10px gold #d4a574, 0.5em tracking. Primary Title: 'Cognitive' in 14vw Boska. Use 'Echo-Layer' effect: three layers of the same word, bottom layer offset -0.04em in color rgba(212, 165, 116, 0.05), middle layer offset -0.02em in color rgba(212, 165, 116, 0.1), top layer #2a2219. Bottom: Max-width 450px text center-aligned. Vertical scroll indicator on bottom left: 1px vertical line 64px tall with 10px vertical text 'SCROLL'.

## Portfolio Grid

Asymmetrical 12-column grid. Large item (7 cols): Aspect 16:10, 32px rounded corners. Small item (5 cols, offset vertically): Aspect 4:5, 32px rounded corners. Each item has a label in 10px gold caps, a title in 3xl Boska, and a 'lucide:arrow-up-right' icon that shifts 4px diagonally on hover.

## Process Section (How It Works)

Horizontal 4-column layout on desktop. Background #fdfcf9 to #f5f1e8 gradient. Connecting line: 1px wide #2a2219/15 across the center. Step indicators: 56px circles, #faf8f3 background, 1px luxury border, text in gold Boska. Card content: 2xl Boska header and sm text in #8c8279. Hover effect: card background turns white with shadow-xl.

## Pricing Tiers

3-card layout. Side cards: White background, luxury border. Middle card (Featured): #2a2219 background, #faf8f3 text, offset -24px vertically. Buttons: Side cards use outlined black borders; Featured card uses #d4a574 background. All cards 2.5rem border-radius. Features listed with #d4a574 checkmarks.

## Footer

Background #2a2219, text #faf8f3. 12-column grid. Left: Large Boska italic logo. Middle: Two columns for navigation links (all-caps gold headers, low-opacity text links). Right: Newsletter box with 16px rounded corners, bg white/5, border white/10. Bottom: Horizontal line with legal text (9px, 0.3em tracking) and social icons in mdi:instagram/twitter/linkedin.

# Special Components

## Echo-Layer Typography

Multi-layered text shadow effect for a high-concept editorial look.

Implementation: Stack three <h1> elements with absolute positioning. Layer 1 (bottom): color rgba(212, 165, 116, 0.05), transform translate(-0.04em, -0.04em). Layer 2 (middle): color rgba(212, 165, 116, 0.1), transform translate(-0.02em, -0.02em). Layer 3 (top): color #2a2219. Apply font-family Boska and negative letter-spacing.

## Luxury Reveal Cards

Interactive cards used for disciplines and features.

Dimensions: variable. Padding: 48px. Background: white/40 with 1px solid rgba(42, 34, 25, 0.08). Hover: bg white, translate-y(-8px), shadow-2xl (shadow color #2a2219 at 0.05 opacity). Icons: 56px circle background (#2a2219 or #d4a574), scale(1.1) on container hover.

## FAQ Accordion

Minimalist interactive Q&A with gold accents.

Container: white, 2rem border-radius, 1px luxury border. Summary: Boska font 24px, cursor-pointer. Icon: 'lucide:plus', rotates 45 degrees on open. Content: Left-border 2px in gold #d4a574, padding-left 24px, text color #8c8279.
